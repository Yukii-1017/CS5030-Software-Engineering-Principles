Aim of the website


The website aims to help developers to understand and improve website accessibility, also known as universal design. We assume you know nothing about website accessibility, but with the help of our website, you will be able to know what is web accessibility and how to use it on your websites.

What is website accessibility?


Put it simply, website accessibility is an index that aims to create â€œdigital resources on the web that everyone can use. '' (Web Accessibility Toolkit, 2013) It is a significant part for all of us. Imagine that when we are old and feel hard to see normal font-size words on screens, a button to adjust the font-size will be thought of considerately.


Improving web accessibility could not only provide convenience for the disabled, kids and the elder, but it also benefits ordinary people. For example, functional screen reading software with accessible websites could allow people to listen to a web without looking at the screen, the caption could not only help to hear disabled people but also help non-native speakers to understand the content of videos.
